# CampusFlow - Smart Attendance System using Face Recognition

This project is deployed on Hugging Face Spaces using a custom Docker container.

## Local Development
To run this application locally on your laptop:
1. Make sure Python 3.12 is installed.
2. Run `start_http.bat` or execute `run.py` using your virtual environment:
   ```bash
   .venv\Scripts\python.exe run.py
   ```
